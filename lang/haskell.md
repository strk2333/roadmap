### Install

Recommand haskell stack

https://docs.haskellstack.org/en/stable/README/#quick-start-guide



```shell

# create new project
stack new my-project
cd my-project
stack setup
stack build
stack exec my-project-exe

# upgrade
stack upgrade

# REPL
stack ghci
```



```haskell
// change prompt
:set prompt "ghci>"

// add module
:module + Data.Ratio
```
